# Rust Ray Tracer
My port of [_Ray Tracing in One Weekend_](https://raytracing.github.io/books/RayTracingInOneWeekend.html) into rust

### Dependencies
In order to compile, you need to install [_cargo_](https://rust-lang.org/learn/get-started/). Once cargo is installed, run:

```bash
cargo run`
```

This will output the image into `images/ch02_gradient.ppm`. This will be in the `.ppm` format so a compatible image viewer is required

### Recommended Image Viewers
- [**gwenview**](https://apps.kde.org/en-gb/gwenview/) - Excellent PPM support for linux (especially KDE Plasma)
- [**Portable Images Viewer**](https://apps.microsoft.com/detail/9nq0n8h413bb?hl=en-US&gl=GB) - Windows image viewer with PPM support
